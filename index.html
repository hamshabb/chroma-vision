Test Instructions

How this test works

This test measures your ability to see patterns based on its color. When a person has color-blindness, they are able to see some colors better than others. By varying the colors of the test images, the type of particular color blindness can be estimated.

Test Format

A series of images will be shown. Look for a number within the dot pattern. Please type the number you see in the space provided below.

Should I guess?

Yes, please feel free to guess what ever number you think is present. It is better to guess than not provide any number at all.
